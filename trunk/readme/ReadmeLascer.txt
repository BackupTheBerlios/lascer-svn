Funktion von Lascer
-------------------

Lascer ist ein Software-System zum Bereich Data-Mining. Informationen zum
Projekt sind über die Homepage http://lascer.berlios.de/ verfügbar.


Verwendung von Lascer
---------------------

Zur Verwendung werden folgende Bibliothek benötigt:

  junit.jar
    http://www.junit.org/
    Getestete Version: 3.8.1

  poi-*.jar
    http://poi.apache.org/
    Getestete Version: 2.5

  weka.jar
    http://sourceforge.net/projects/weka/
    Getestete Version: 3.4.7

  commandline.jar
    http://architeuthis.berlios.de/
    Getestete Version: 2.0.1

  mathCollection.jar
    http://mathcollection.berlios.de/
    Getestete Version: 1.3.3

  archiUser.jar
    http://architeuthis.berlios.de/
      Im Paket testenvironment-* als Datei "User.jar" vorhanden.
    Getestete Version: 2.0.1

  mengenUeberdeckung.jar
    http://lascer.berlios.de/
      Teilprojekt MengenUeberdeckung.
    Getestete Version: 1.0

  coreExtended.jar
    http://lascer.berlios.de/
      Teilprojekt WekaExtention.
    Getestete Version: 1.0

  geomClusterung.jar
    http://lascer.berlios.de/
      Teilprojekt Clusterung.
    Getestete Version: 1.0

Die Verwendung von Lascer ist in der Datei evalScripts/beschreibung.txt
beschrieben.
