Funktion vom Paket egtb
-----------------------

Das Paket egtb (end game tablebase) dient der Konvertierung einer Datei
mit den analysierten Daten eines Endspiels in eine Datei im ARFF-Format.


Verwendung vom Paket egtb
-------------------------

Zur Erzeugung der analysierten Daten kann man die Programme espanalyse3
und espanalyse4 im Paket "endspiel" verwenden. Zur Konvertierung ist die
dann Klasse "Conversion" auszuführen. Beim Aufruf ohne Parameter erhält
man Hinweise zur Benutzung.
