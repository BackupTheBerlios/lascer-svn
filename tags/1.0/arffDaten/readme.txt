                        Dateien in diesem Verzeichnis


Die Dateien in diesem Verzeichnis beschreiben die Daten, die zum Test von
Lascer verwendet wurden und einzelne weitere Datensätze, sowie ihre
Erzeugung. Die Daten sind im arff-Format
(http://www.cs.waikato.ac.nz/~ml/weka/arff.html) vorhanden, das von Weka
verwendet wird. Zu jedem Datensatz ist eine info-Datei vorhanden. Die
zugehörigen Datensätze liegen im FTP-Bereich (unter
ftp://ftp.berlios.de/pub/lascer/arffDaten/).

Zum Compilieren der Pascal-Programme wird in den info-Dateie der Compiler
von "Free Pascal" (fpc) verwendet. Bei Verwendung des Compilers von "GNU
Pascal" (gpc) ist als Option "-o execdatei" anzugeben, wobei "execdatei" der
Name des Programms ohne die Endung ".pp" bzw. ".pas" ist.
